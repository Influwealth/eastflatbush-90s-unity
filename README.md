## Quick Start Instructions for Youth Developers and AI Agents
- **Make sure you have Unity installed.**
- Clone the repository using:
  ```bash
  git clone https://github.com/Influwealth/eastflatbush-90s-unity.git
  cd eastflatbush-90s-unity
  ```
- Open the project in Unity.
- Follow the setup instructions in the project!

## Technical Architecture (CCA Model)
The project is structured using the CCA Model, which includes:
- **Components**: Modular building blocks of the game.
- **Controllers**: Manage game state and input.
- **Agents**: Enforce behavior of non-player characters (NPCs).

## WealthBridge OS Integration
- Our game integrates with the WealthBridge OS to ensure the best performance and engagement for players.
- Ensure the WealthBridge module is configured properly in the settings.

## Deployment Instructions for WebGL and Standalone Builds
- **WebGL Deployment**:
  1. In Unity, go to `File > Build Settings`.
  2. Select `WebGL` as the build platform.
  3. Click `Build` and choose a location to save.
- **Standalone Build**:
  1. Same as above, just choose `PC, Mac & Linux Standalone` instead.
